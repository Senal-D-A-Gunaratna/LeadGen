# **App Name**: LeadGen

## Core Features:

- Real-Time fingrprint Attendance Tracking: Scans student finger via fingrprint sensor node and updates attendance records live.
- Comprehensive Student Profiles: Maintains a database of student profiles, excluing photos, grades, personal information, and attendance history. Uses mock data for the MVP.
- Live Attendance Dashboard: Displays current attendance status with present/absent indicators in real-time.
- Average Attendance Statistics: Generates visual analytics and trends for average attendance.
- Class-wise and Grade-wise Attendance: Provides breakdowns of attendance by class and grade.
- Individual Student Attendance History: Allows viewing of a student's attendance history in a calendar format.
- Predictive Attendance Tool: Uses generative AI as a tool to predict potential low attendance based on historical trends and external factors.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a futuristic, high-tech feel.
- Background color: Dark gray (#292929) for a dark theme, allowing the neon accents to stand out.
- Accent color: Vibrant purple (#BE29EC) for highlighting interactive elements and key data points.
- Headline font: 'Space Grotesk', a sans-serif font with a computerized look; body font: 'Inter', a grotesque-style sans-serif for readability.
- Use sleek, minimalist icons inspired by futuristic UI designs. Icons will feature neon glows and thin line weights.
- Employs a glassmorphism effect with frosted glass cards and floating UI elements. The layout is responsive and adapts to different screen sizes.
- Smooth transitions and pulsing animations are used for fingrprint scanning, progress indicators, and data updates. Includes particle effects for background ambience.
